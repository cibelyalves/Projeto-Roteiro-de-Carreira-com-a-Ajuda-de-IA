🚀 PROJETO DE PORTFÓLIO
PROJETO: Assistente de Atendimento Inteligente para Serviços Financeiros
O QUE FAZER:
Criar um assistente com IA que recebe perguntas de clientes, classifica a solicitação (ex: cartão, empréstimo, limite), gera resposta automática e organiza os dados em uma planilha para análise.
ENTREGÁVEIS:

Biblioteca de prompts de atendimento
Fluxo automatizado (ex: IA → Planilha)
Dashboard simples com dados dos atendimentos

CRITÉRIOS DE ACEITAÇÃO:

IA classifica corretamente as solicitações
Respostas automatizadas claras e úteis
Dados organizados para análise posterior

DICA: Use sua experiência com atendimento bancário para simular perguntas reais de clientes.

💬 ROTEIRO DE ENTREVISTAS
PERGUNTA 1: Por que você está migrando da área bancária para IA?
COMO RESPONDER:
Explique que já atuava resolvendo problemas de clientes e agora busca escalar esse impacto com automação inteligente.
PERGUNTA 2: O que é Prompt Engineering?
COMO RESPONDER:
É a prática de estruturar instruções para que modelos de IA gerem respostas úteis e consistentes.
PERGUNTA 3: Fale sobre seu projeto
COMO RESPONDER:
Descreva como automatizou a triagem de solicitações e organizou dados para tomada de decisão.
PERGUNTA 4: Como a IA pode ajudar no atendimento ao cliente?
COMO RESPONDER:
Reduz tempo de resposta, melhora consistência e permite análise de padrões.
PERGUNTA 5: Como você garante qualidade nas respostas da IA?
COMO RESPONDER:
Testando diferentes prompts e ajustando conforme resultados.
